# Smart Planner & Burnout Prevention System

## Project Description
<br />
Mindful Study, A Study Planner and Burnout Prevention System, is a web-based platform designed to help students manage their academic responsibilities while maintaining a healthy study-life balance. Many students struggle with organizing assignments, tracking deadlines, balancing personal schedules, and avoiding academic burnout due to heavy workloads and poor time management. This system aims to provide an organized and user-friendly solution that supports both productivity and student well-being.

The platform allows users to create study schedules, manage tasks and assignments, track academic progress,and receive reminders for upcoming deadlines. In addition, the system includes burnout prevention features such as study-break recommendations, workload monitoring, and productivity analysis to encourage healthier study habits.

This is a cross-over project among 4 courses. It integrates the concepts from Human-Computer Interaction, Software Engineering, Database Administration, and Backend Development.
<br />

## Objectives
<br />
The objectives of this project are:
1. To help students effectively manage study schedules, assignments, and deadlines.
2. To provide tools that encourage healthier study habits and reduce academic burnout.
3. To design a user-friendly and accessible interface based on HCI principles.
4. To implement a functional backend system with secure user authentication and data management.
5. To develop a structured database system for storing user tasks, schedules, and productivity records.

## Requirements
<br />

### Human-Computer Interaction (HCI)
Target user: University Students who desire a balancing academic life and personal life
• Pain point: Poor time management,
• User story: As a student, I want to create and manage study tasks while monitoring my stress level so that I can avoid burnout and complete assignments on time.
• Screens: Authentication (Login/Signup), Dashboard, Planner, Analytics, Settings
• Navigation/feedback: Clear authentication flow, dashboard navigation, back
navigation between screens, task creation confirmations, burnout assessment
results, study plan recommendations, Visible task status, and upcoming deadlines.

### Software Engineering
#### Functional
• Add and manage tasks
• Create Study Session
• Record mood
• Display Insights
• Recommend break
• Manage Settings
#### Non-functional
• Easy to use
• Fast response
• Simple navigation

### Database Administration
• Normalize Tables or Entities to 3rd normal form
• Generation of million records of data for Sample
• Create queries that are meaningful for the system
• Create users’ role and privileges

### Backend Development
Implement secure user authentication and authorization
• Provide task and deadline management features
• Allow students to schedule study sessions
• Monitor stress and burnout indicators
• Generate dashboard insights regarding academic progress
